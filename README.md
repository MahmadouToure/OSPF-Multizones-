# OSPF-Multizones-

## Description 
Projet réalisé sur Cisco Packet Tracer simulant plusieurs zones dans un réseau utilisant le protocole de routage OSPF       

## Objectifs
Permettre la communication entre tous les routeurs         

## Équipements utilisés
9 Router 2911      

## Configuration principale
Mise en place du protocole sur les routeurs       
Configuration des réseaux, masques inversés et des zones sur chaque routeurs       
Vérification des tables de routage         
Test de connectivité
